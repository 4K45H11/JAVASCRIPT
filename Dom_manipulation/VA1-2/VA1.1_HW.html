HTML Head Tag

The head Element

The head element is container for metadata(data about data) and is placed between the html tag and the body tag.

The title Element

The title element defines the title of the document.It is shown in the browser's title bar or in a page's tab.

The Page title is used by search engine algorithms to decide the data while listing pages in the search result.

The link element

The link elelment defines the relationship between the current document and an external resource.The link tag is most often used to link external style sheets.

The meta element

The meta element is typically used to specify the character set, page description, keywords, author of the document and viewport settings.The metadata will not be displayed on the page, but is used by (how to display content), by search engines for keywords and other web services. Some of the meta tags are:

  • Defines the character set used:

    <meta charset="UTF-8">

  • Defines the viewport to make your website look good on all devices.

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

  • Defines keywords for search engine.

    <meta name="keywords" content="HTML,CSS,JavaScript">

  • Defines author of the website

    <meta name="author" content="John Doe">
